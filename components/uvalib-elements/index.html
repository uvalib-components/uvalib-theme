Library Web Components

  • uvalib-alerts (demo)

    This component binds to the Library Alerts feed and provides the UI for viewing Library Alerts on the Library web site.

  • uvalib-analytics (demo)

    This component setsup web tracking for your site given a site id. It is helpful in tracking the state of single page apps and also helps track events.

  • uvalib-card (demo)

    This component creates paper-cards that are bound to the Library Resource Card API.

  • uvalib-footer (demo)

    This component sets up the Library's standard footer with a custom bottom bar option.

  • uvalib-header (demo)

    This component sets up the Library's standard header.

  • uvalib-liame (demo)

    This component is an email obfuscator. It has a number of options but basicaly takes a string which it will reverse and then concatenate @virginia.edu to make a link. It hides the email from spam bots with the hope that most are fairly simple and don't go to extraordinary lengths.

  • uvalib-logos (demo)

    This component provides a UVA Library logo in all of it's optional layouts/styles.
